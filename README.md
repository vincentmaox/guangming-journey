# 光明西游·攻略记

深圳光明区多人协作回合制RPG游戏。基于 Socket.IO 模板开发，支持 1-4 人组队，探索虹桥公园、科技馆、美术馆、回归亭、光明侨院等光明区特色地点，组团完成光明攻略！

## 游戏特色

- **大话西游风格回合制RPG**：经典回合制战斗，策略搭配队伍阵容
- **7大特色角色**：李博士(法师)、Tony(辅助)、小琪(治疗)、晨光(战士)、老周(坦克)、阿珍(增益)、荔宝(刺客)
- **5大光明关卡**：虹桥公园 → 科技馆 → 美术馆 → 回归亭 → 光明侨院（最终BOSS）
- **多人协作**：支持 1-4 人组队，角色不可重复，团队配合通关
- **丰富的技能系统**：每个角色拥有2个独特技能，涵盖伤害、治疗、增益、减益、复活等

## 本地运行

```bash
npm install
npm start
```

访问 http://localhost:3000

## Docker 运行

```bash
docker build -t guangming-journey .
docker run --rm -p 3000:3000 guangming-journey
curl http://localhost:3000/health
```

## 游戏流程

1. **创建/加入房间**：单人试玩或创建多人房间，邀请好友
2. **选择角色**：7个光明IP角色任选，每个角色有独特定位和技能
3. **攻略地图**：房主选择关卡，按顺序攻略光明五大景点
4. **回合制战斗**：按速度排序行动，使用普攻/技能/防御击败敌人
5. **通关结算**：胜利后恢复状态，继续攻略下一关，直至击败侨院之主

## 技术栈

- Node.js + Express + Socket.IO
- 纯前端 Canvas/CSS 渲染
- 支持移动端操作
